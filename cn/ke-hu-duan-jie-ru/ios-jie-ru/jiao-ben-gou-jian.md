# 脚本构建

## 1.cd当前工程目录

```text
cd /User/XXXX
```

## 2.下载Vigame资源

## 2.1 执行clone命令下载

```text
git clone https://github.com/hohua88/Vigame.git
```

## 2.2 手动下载

[地址](https://github.com/hohua88/Vigame) 



## 3.执行auto.sh脚本

```text
./Vigame/auto.sh a03cfead89f5934d #执行脚本 参数为singleid是必须
```

脚本执行完成后会上传到fir.im，之后会打开打包的文件夹，里面有个二维码图片，可以把这个图片给到测试下载。
